title: 'Campaign Performance'
api: 'GET /api/public/performance/?date_from={date_from}&date_to={date_to}&advertiser_id={advertiser_id}'
---

<Note>
This endpoint is protected and requires an agency API key for access.
</Note>

<RequestExample>
```bash Request
  curl --request GET \
    --url http://gc-api-prod.us-east-1.elasticbeanstalk.com/api/public/performance/?date_from=2025-01-01&date_to=2025-1-03&advertiser_id=544
```
</RequestExample>

## Headers

<ResponseField name="X-Api-Key" type="string" required>
  The agency's API key for authorization ask admin to create ot for your usage
</ResponseField>

## Parms

<ParamField path="date_from" type="string" example="2025-01-01" required>
  Date staring from response will be generated <code>?date_from=2025-01-01</code> 
</ParamField>
<ParamField path="date_to" type="string" example="2025-01-01" required>
  Date ending with response will be generated <code>?date_to=2025-01-03</code> 
</ParamField>
<ParamField path="advertiser_id" type="number" example="544" required>
  The unique identifier for the adveticer <code>?advertiser_id=544</code> 
</ParamField>
